# asistencia-indi
Sistema de gestion de entradas y salidas de una institucion educativa salvadoreña 
Este sisrema pertende ser una solucion de autoamtizacion par el ingreso y salida de estudiantes a la institucion educativa, su valor agreagdo consiste en que notifica
inmediatamente al padre de familia o responsable del ingreso o salida de su hijo de la insitucion
